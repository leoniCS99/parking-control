# Projeto parking-control
Aplicação que realiza a Gestão de Controle para Vagas de Estacionamento


## Tecnologias

Tecnologias utilizadas neste projeto.

* Java 11
* Spring Boot 2.3.6
* Spring Web
* Spring Data JPA
* Spring Validation
* Postgres


## Serviços Usados

* Github
* Git
